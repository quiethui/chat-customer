## 核心术语统一

为准确反映项目架构，以下术语已全局统一：

| 旧术语 | 新术语 | 说明 |
|---|---|---|
| `user` / `用户` | `customer` / `客户` | 终端用户，通过 widget 挂件咨询 |
| `user` / `用户` | `manager` / `坐席/管理员` | 客服人员，通过 web 管理端登录 |
| `user_id` | `customer_id` | 客户 ID，用于数据隔离 |
| `get_current_user` | `get_current_customer` / `get_current_manager` | 依赖注入函数 |
| `get_current_admin_user` | `get_current_admin_manager` | 管理员权限校验 |
| Vue 前端 `web/` | 管理端 `web/`（React） | 已完成迁移，旧 Vue 代码已废弃 |
| 前端（建设中） | 客户端挂件 `widget/` | 已实现并投入使用 |
